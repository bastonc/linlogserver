All function in last version:

  • Entry QSO to base
  • Search repeat QSO
  • Edit QSO (click to cell -> edit information -> press Enter(or click to other cell) -> press Enter(save all edit cells))
  • Send eQSL (need account eqsl.cc) after add QSO to base
  • Cluster with autofill (by click on spot) and filter by:
    • band
    • spot (prefix)
    • spotter (prefix)
  • Search image by call in qrz.com base
  • Implemented TCI protocol for driving radiostation

How to install

  1. Download installer
  2. Set executable bit ("chmod -X name installer file" or use GUI - Mouse Right Button -> "Premissions" -> checkbox "Executable")
  3. Run installer (recomended execute in terminal - if you have errors - please take screenshot errors and send to my email - bastonsv@gmail.com)

After installation run LinLog from menu DE

LinLog and installer tested into Ubuntu, Kubuntu, Mint Cinnamon, Fedora Workstation, Fedora KDE (spins)



System required for version 1.1 +:

OS GNU/Linux: Fedora WS, CentOS, Red-hat, Debian, Mint, Ubuntu, Manjaro, MX Linux



System required before version 1.1:
  • Operating system: GNU/Linux
  • Python 3.7.3 +
  • GLIBC_2.29
  • Wayland
  • RAM 100 Mb and higer. RAM is using in direct ratio with quantity of records in your Log (in log.adi file).
    • If your system does not have these dependencies, please download an alternative version for your system (only v1.01).
